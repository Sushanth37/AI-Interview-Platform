# AI Interview Preparation Platform

An AI-powered interview preparation platform built using Flask, MySQL, Bootstrap, and Groq AI.

## Features

- Resume Upload and Parsing
- AI Resume Analysis
- AI Interview Question Generation
- Mock Interview Simulation
- AI-Based Performance Feedback
- MySQL Interview History Dashboard

## Tech Stack

- Python
- Flask
- MySQL
- Bootstrap 5
- Groq API
- HTML/CSS

## Project Workflow

1. Upload Resume
2. AI Analyzes Resume
3. Generate Personalized Interview Questions
4. Attend Mock Interview
5. Receive AI Feedback
6. Store Results in MySQL Database
7. View Historical Performance Dashboard
